<h1 align="center">Desenvolvimento de Sistemas</h1>
<p>Neste repositório ficará armazenado todas as aulas, arquivos e exercício de <b>Desenvolvimento de Sistema</b>, Uma matéria onde usamos <b>Java</b> para criar programas e interfaces, conhecimentos que usaremos para solucionar problemas do dia a dia</p>
<br>
<p>Sobre o <b>Java</b>, ele é uma linguagem orientada a objetos, o que permite você criar classes com atributos e métodos e usá-los na classe principal, o que torna o código mais prático e eficiente</p>
